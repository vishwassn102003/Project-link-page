About This Project

This internship project is a complete Face Recognition Check-In System. It includes:

• Face Registration – Users register by capturing their face using the camera.
• Face Identification – The system recognizes the user and marks check-in/check-out.
• Admin Dashboard – Admin can manage users and create events for which check-ins are recorded.

The backend is powered by Python Flask + AWS Rekognition, deployed on Render Cloud.
